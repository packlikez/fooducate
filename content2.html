แล้วจะรู้ได้อย่างไร ว่าในแต่ละวันเราต้องการพลังงานมากน้อยแค่ไหน?

November 22, 2017

แล้วจะรู้ได้อย่างไร ว่าในแต่ละวันเราต้องการพลังงานมากน้อยแค่ไหน จะได้ไม่เหลือพลังงานส่วนเกิน เรามีสูตรคำนวณพลังงานแบบง่าย ๆ มาให้ลองคำนวณกัน สำหรับวิธีคำนวณพลังงานในแต่ละวัน มีอยู่ด้วยกันหลายสูตร แต่ขอนำเสนอสูตรที่นิยมใช้กันมากที่สุด คือ

Basal Metabolic Rate (BMR)

เป็นสูตรคำนวณพลังงานที่ร่างกายต้องการในแต่ละวัน เพื่อให้ระบบต่าง ๆ ในร่างกายสามารถทำงานได้ในขณะที่ไม่ได้ทำกิจกรรมอะไร สามารถคำนวณได้ดังนี้

BMR สำหรับผู้ชาย = 66 + (13.7 x น้ำหนักตัว (กิโลกรัม))+(5 x ส่วนสูง (เซนติเมตร))-(6.8 x อายุ)

BMR สำหรับผู้หญิง = 665 + (9.6 x น้ำหนักตัว (กิโลกรัม))+(1.8 x ส่วนสูง (เซนติเมตร))-(4.7 x อายุ)

สมมติว่า A เป็นผู้หญิง อายุ 30 ปี ส่วนสูง 165 ซม. น้ำหนัก 60 กก.
BMR จะเท่ากับ 665 +(9.6 x 60)+(1.8 x 165)-(4.7 x 30) = 1,397 กิโลแคลอรี หมายถึงว่า A ต้องการพลังงานอย่างน้อย 1,397 กิโลแคลอรี ในการดำรงชีวิตในแต่ละวันในกรณีที่วันนั้น A ไม่ได้ทำกิจกรรมอะไรเลย

อย่าลืมว่าในแต่ละวันเราต้องทำกิจกรรมมากมาย ทั้งนั่ง เดิน นอน ดูทีวี ทำงาน ออกกำลังกาย ซึ่งกิจกรรมของแต่ละคนไม่เหมือนกัน พลังงานที่ต้องการในแต่ละวันจึงไม่เท่ากันด้วย ดังนั้นเราต้องมาคำนวณหาปริมาณแคลอรีที่เราใช้ในแต่ละวัน

วิธีการก็คือ ดูว่าในแต่ละวันเราทำกิจกรรมและออกกำลังกายมากน้อยแค่ไหน แล้วนำค่า BMR ที่ได้ไปคูณดังนี้

นั่งทำงานอยู่กับที่ และไม่ได้ออกกำลังกายเลย หรือน้อยมาก = BMR x 1.2
ออกกำลังกายหรือเล่นกีฬาเล็กน้อย 1-3 วัน/สัปดาห์เดินบ้างเล็กน้อย ทำงานออฟฟิศ = BMR x 1.375
ออกกำลังกายหรือเล่นกีฬาปานกลาง 3-5 วัน/สัปดาห์เคลื่อนที่ตลอดเวลา = BMR x 1.55
ออกกำลังกายหรือเล่นกีฬาอย่างหนัก 6-7 วัน/สัปดาห์ = BMR x 1.725
ออกกำลังกายหรือเล่นกีฬาอย่างหนัก หรือเป็นนักกีฬา ทำงานที่ใช้แรงงานมาก = BMR x 1.9

เช่น คุณ A มีค่า BMR = 1,397 กิโลกรัมแคลอรี เป็นคนที่แทบไม่ได้ออกกำลังกายเลย จะได้ปริมาณแคลอรีที่ต้องการในหนึ่งวันเป็น 1,397x1.2 = 1,676.4 ดังนั้นถ้าต้องการควบคุมน้ำหนัก ก็ไม่ควรทานอาหารมากเกินกว่า 1,676.4 กิโลแคลอรี
